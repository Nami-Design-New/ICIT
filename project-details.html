Taxi Booking App

منصة مركز التسوق للحصول على معلومات المحلات التجارية والترقيات. تم التنفيذ بناءً على التصميم المقدم نيابة عن JU: Agencja. استخدم WordPress كنظام إدارة محتوى (CMS) مع موضوع تم إنشاؤه من الصفر.

Technologies Used
  • Flutter
  • Dart
  • Firebase
  • Google Maps API
  • Cotlin
  • Java
Live Preview

الابداع المتكامل لتقنية المعلومات هي شركة تطوير برمجيات ديناميكية ومبتكرة. نحن متخصصون في إنشاء حلول متطورة تدفع النمو والكفاءة والنجاح للشركات عبر الصناعات.
